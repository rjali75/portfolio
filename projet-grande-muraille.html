Site web Grande Muraille de Chine

Site web éducatif et responsive sur le patrimoine mondial

Présentation du projet

Création d'un site web complet et responsive dédié à la Grande Muraille de Chine, l'une des plus grandes merveilles architecturales du monde. Le projet vise à présenter l'histoire, l'architecture et la signification culturelle de ce monument emblématique tout en respectant strictement la charte graphique M.C.N.

Le site a été développé en HTML/CSS pur, sans framework, démontrant une maîtrise des fondamentaux du développement web et du responsive design. L'accent a été mis sur l'accessibilité, la performance et l'expérience utilisateur sur tous les appareils.

Objectifs du projet

1

Respecter la charte graphique M.C.N.

Application stricte des couleurs, typographies et espacements définis dans la charte graphique fournie.

2

Design responsive

Adaptation parfaite sur tous les supports (desktop, tablette, mobile) avec breakpoints optimisés.

3

Contenu éducatif de qualité

Présentation claire et structurée de l'histoire et des caractéristiques de la Grande Muraille.

4

Navigation intuitive

Structure de navigation claire permettant une exploration fluide du contenu.

Caractéristiques techniques

Responsive Design

Media queries CSS pour adaptation sur mobile (320px+), tablette (768px+) et desktop (1024px+).

Charte graphique M.C.N.

Respect strict des couleurs, typographies et espacements définis. Variables CSS pour cohérence.

Performance

Images optimisées, CSS minifié, chargement rapide. Pas de dépendances externes lourdes.

Accessibilité

HTML sémantique, attributs alt sur images, contraste respecté, navigation au clavier.

Navigation

Menu fixe, fil d'Ariane, liens internes, retour haut de page. UX optimisée.

Multimédia

Galerie d'images, contenu vidéo intégré, cartes interactives, animations CSS.

Défis techniques

1. Responsive design sans framework

Création d'un système de grille personnalisé et gestion des breakpoints sans Bootstrap ou autre framework. Utilisation de CSS Grid et Flexbox pour des layouts flexibles et maintenables.

2. Optimisation des images

Compression et redimensionnement des images pour performances optimales tout en conservant la qualité visuelle. Utilisation de formats modernes (WebP) avec fallbacks.

3. Respect strict de la charte

Application précise de toutes les directives de la charte graphique M.C.N. (couleurs, typographies, espacements) tout en maintenant une flexibilité pour le responsive.

4. Accessibilité web

Garantir que le site soit accessible à tous (WCAG 2.1 niveau AA) : contrastes suffisants, navigation clavier, lecteurs d'écran, textes alternatifs.

Résultats

Le site web sur la Grande Muraille de Chine a été validé avec succès, démontrant une excellente maîtrise des fondamentaux du développement web. Le respect scrupuleux de la charte graphique M.C.N. et l'implémentation d'un design entièrement responsive ont été particulièrement appréciés.

Ce projet m'a permis de consolider mes compétences en HTML/CSS et de comprendre l'importance du respect des contraintes graphiques dans un contexte professionnel. La création d'un site sans framework a renforcé ma compréhension des mécanismes fondamentaux du web.
