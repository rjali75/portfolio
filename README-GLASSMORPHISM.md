# 🌌 Portfolio Ali Khaloui - Glassmorphism Pro

## Thème appliqué : Glassmorphism Pro

Tu as maintenant un portfolio avec le design **Glassmorphism Pro**, un thème moderne et élégant inspiré par les interfaces Apple et les tendances 2025.

## 🎨 Caractéristiques du thème

### Design
- **Effet de verre** : Tous les éléments ont un effet de transparence avec flou d'arrière-plan (backdrop-filter)
- **Dégradés violets/bleus** : Palette élégante (#667eea → #764ba2)
- **Animations fluides** : Transitions douces et naturelles
- **Ombres subtiles** : Effets de profondeur élégants

### Couleurs
- **Principal** : #667eea (Bleu-violet)
- **Secondaire** : #764ba2 (Violet profond)
- **Accent** : #f093fb (Rose-violet)
- **Fond** : Dégradé pastel (#e0c3fc → #8ec5fc)

### Typographie
- **Font** : Poppins (moderne, arrondie, friendly)
- **Poids** : De 300 (Light) à 900 (Black)

### Effets spéciaux
- ✨ Glassmorphism sur tous les cards
- 🌊 Fond animé avec dégradés qui bougent
- 🎯 Hover effects dynamiques
- 💫 Micro-animations partout
- 🔮 Bordures semi-transparentes

## 📁 Fichiers inclus

- `index.html` - Page principale
- `style-glassmorphism.css` - CSS principal avec le thème
- `project-style-glassmorphism.css` - CSS pour les pages de projets

## 🚀 Comment utiliser

1. **Extrais le ZIP**
2. **Ouvre `index.html`** dans ton navigateur
3. **Admire le résultat** ! 🎉

## 🔄 Revenir au thème classique

Si tu veux revenir au thème sobre original :

1. Dans `index.html`, change :
   ```html
   <link rel="stylesheet" href="style-glassmorphism.css">
   ```
   par :
   ```html
   <link rel="stylesheet" href="style.css">
   ```

2. Dans toutes les pages `projet-*.html`, change :
   ```html
   <link rel="stylesheet" href="style-glassmorphism.css">
   <link rel="stylesheet" href="project-style-glassmorphism.css">
   ```
   par :
   ```html
   <link rel="stylesheet" href="style.css">
   <link rel="stylesheet" href="project-style.css">
   ```
