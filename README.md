# Portfolio - Ali Khaloui

Site personnel d'Ali Khaloui, étudiant en BUT Informatique à l'IUT de Marne-la-Vallée (Université Gustave Eiffel). Présente le parcours, les compétences et les projets réalisés dans le cadre de la recherche d'une alternance en développement.

## Fichiers

- `index.html` — Page d'accueil (hero, à propos, compétences, projets, contact)
- `projet-wall-is-you.html` — Projet Python, jeu dungeon-crawler
- `projet-chateau-quest.html` — Projet PWA, Château de Pierrefonds
- `projet-grande-muraille.html` — Projet web, site sur la Grande Muraille
- `projet-base-donnees.html` — Projet SQL, centre médical
- `style.css` — Styles principaux
- `project-style.css` — Styles des pages projets
- `script.js` — Navigation, scroll, animations

## Technos

HTML5, CSS3 (glassmorphism, variables CSS, responsive design), JavaScript vanilla.
Police Poppins via Google Fonts. Logos des techs via simpleicons.org.
