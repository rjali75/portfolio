## 🚀 Installation et utilisation

### Étape 1 : Extraire les fichiers
Extrayez le fichier ZIP dans un dossier de votre choix.

### Étape 2 : Remplacer la photo de profil
1. Préparez une photo de profil carrée (recommandé : 500x500px minimum)
2. Nommez-la `profile.jpg` ou `profile.png`
3. Remplacez le placeholder dans le HTML :

```html
<!-- Recherchez cette section dans index.html -->
<div class="image-placeholder">
    <!-- Remplacez ce SVG par votre image -->
    <img src="profile.jpg" alt="Ali Khaloui">
</div>
```

### Étape 3 : Personnaliser (optionnel)
Vous pouvez personnaliser :
- Les couleurs dans `style.css` (variables CSS au début du fichier)
- Les informations personnelles dans `index.html`
- Les projets et compétences

### Étape 4 : Publier en ligne

#### Option 1 : GitHub Pages (gratuit)
1. Créez un compte GitHub si ce n'est pas déjà fait
2. Créez un nouveau repository nommé `votre-username.github.io`
3. Uploadez tous les fichiers du portfolio
4. Activez GitHub Pages dans les paramètres du repository
5. Votre site sera accessible sur `https://votre-username.github.io`

#### Option 2 : Netlify (gratuit)
1. Créez un compte sur [Netlify](https://www.netlify.com/)
2. Glissez-déposez le dossier du portfolio dans Netlify
3. Votre site sera publié automatiquement

#### Option 3 : Vercel (gratuit)
1. Créez un compte sur [Vercel](https://vercel.com/)
2. Importez votre projet depuis GitHub ou uploadez les fichiers
3. Déployez en un clic

## 🎨 Personnalisation des couleurs

Pour changer les couleurs, modifiez les variables CSS dans `style.css` :

```css
:root {
    --color-primary: #2563eb;        /* Bleu principal */
    --color-primary-dark: #1e40af;   /* Bleu foncé */
    --color-accent: #f59e0b;         /* Orange accent */
    --color-text: #1f2937;           /* Texte principal */
    --color-text-light: #6b7280;     /* Texte secondaire */
}
```
